Recursividade em Java – Introdução às Estruturas de Dados

Enunciado:
Criar uma aplicação em Java com uma **função recursiva** que receba dois valores inteiros, **dividendo** e **divisor**, e calcule o **resto da divisão inteira usando apenas subtrações**. A função deve realizar a operação de forma recursiva até atingir a condição de parada.
Os valores de entrada devem ser números inteiros positivos.


Objetivo

O objetivo deste exercício é praticar:
* Recursividade em Java
* Operações com números inteiros
* Identificação de condição de parada em funções recursivas
* Relação entre o valor atual e o valor anterior na construção da solução recursiva
* Raciocínio lógico aplicado a operações matemáticas

Requisitos
O programa deve:
1. Receber dois números inteiros positivos (**dividendo** e **divisor**) como parâmetros.
2. Calcular o **resto da divisão inteira** usando apenas subtrações e recursão.
3. Não utilizar laços (`for`, `while`) para calcular o resto (exceto para tratamento de entrada).
4. Retornar o resultado como número inteiro (`int`).
5. Validar a entrada para garantir que ambos os números sejam positivos e que o divisor seja diferente de zero.
6. Incluir tratamento de exceção (`try-catch`) para entradas inválidas.

Tecnologias
* Oracle Java
* Programação Orientada a Objetos (POO)
* Recursividade

Observações:
Este exercício faz parte dos estudos de **Introdução às Estruturas de Dados**, com foco no desenvolvimento do raciocínio lógico e na compreensão de funções recursivas aplicadas a operações matemáticas básicas.
